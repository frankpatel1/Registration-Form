# Registration Form

## 📝 Overview

Registration Form is a responsive web application designed to collect and validate user information through a clean and user-friendly interface. Built with HTML, CSS, and JavaScript, the project showcases essential front-end development concepts such as form validation, user input handling, error messaging, and responsive design.

This project serves as an excellent example for developers learning how to create professional registration forms and improve user experience through client-side validation techniques.

## ✨ Features

* User registration form interface
* Real-time form validation
* Required field verification
* Email format validation
* Password input support
* Error handling and feedback messages
* Responsive design for all devices
* Clean and modern UI
* Lightweight and easy to customize

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Registration-Form.git
```

Navigate to the project directory:

```bash
cd Registration-Form
```

Open the application:

```bash
index.html
```

Or simply double-click `index.html` to launch it in your browser.

## 📖 Usage

1. Open the registration form in your browser.
2. Enter the required information.
3. Complete all mandatory fields.
4. Submit the form.
5. View validation messages if any errors occur.
6. Successfully register after all validations pass.
